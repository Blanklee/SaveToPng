# SaveToPng
- 클립보드에 있는 이미지를 .png로 저장하는 프로그램
- SRS7 등의 WinPE 64bit 상에서 화면을 캡처하는 것이 목적임
- WinClon, SRS5 에서도 32bit로 실행 가능
- PrtScr키 누르고 SaveToPng에 붙여넣은 후 저장하면 됨
- 배포: WinPE 상에서 실행할때 shfolder.dll를 함께 배포할 것
